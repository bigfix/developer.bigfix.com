--- title: Evaluate Relevance ---

In this section you can test your relevance expressions by typing them in the text box and clicking Evaluate.

The evaluation is run on a Linux CentOS system. Ensure that you run the relevance expressions that are supported on that type of operating system.

For example, if you click Evaluate, the expression name of operating system is run and the information about the operating system is returned.
